# Instruction
- Clone the project
- Cd /jobApp folder
- Execute: npm install
- Execute: npm run build
- Execute: node server
- Access http://localhost:9000
- Sample Employeer account / password: employeer1 / 123456 
- Sample Job seeker account / password: seeker1 / 123456
